# NODNOL - Addiction Recovery Support Platform

> A comprehensive web application supporting individuals recovering from crystal methamphetamine addiction, along with powerful tools for healthcare professionals and case workers.

## 🎯 Project Overview

NODNOL is an innovative, AI-powered platform designed to provide comprehensive support for individuals struggling with crystal methamphetamine addiction. The platform combines evidence-based recovery practices with cutting-edge technology to create a supportive ecosystem for both clients and healthcare professionals.

### Key Features
- 🚨 **Crisis Support**: 24/7 emergency assistance and safety tools
- 🤖 **AI-Powered**: Intelligent features for personalized recovery support
- 👥 **Community**: Anonymous peer support and professional networking
- 📱 **Accessible**: Responsive design with privacy and anonymity options
- 🔒 **Secure**: HIPAA-compliant with end-to-end encryption
- 🎨 **Modern UI**: Futuristic, 3D interface with accessibility features

## 🚀 Quick Start

### For Clients (Recovery Seekers)
1. Visit the platform (anonymous access available)
2. Complete optional privacy-focused onboarding
3. Access crisis support, recovery tools, and community features
4. Connect with professional help when ready

### For Healthcare Professionals
1. Request professional access credentials
2. Complete staff training and certification
3. Access case management tools and client dashboards
4. Begin supporting clients through the platform

## ✨ Features

### For Individuals in Recovery

#### Crisis Support & Safety
- Emergency contact system with one-click access
- Crisis detection and intervention algorithms
- Guided grounding exercises for managing symptoms
- Safety planning tools with personalized strategies

#### Recovery Tools
- Memory aids and cognitive support features
- Daily mood and symptom tracking with AI analysis
- Recovery diary with voice-to-text capabilities
- Progress visualization and milestone tracking

#### Community & Support
- Anonymous peer support chat rooms
- Virtual recovery meetings and events
- Goal setting with community accountability
- Resource sharing and success stories

#### Privacy & Accessibility
- Complete anonymity options (no registration required)
- Discreet UI mode for public use
- Voice control and accessibility features
- Customizable themes and layouts

### For Healthcare Professionals

#### Case Management
- Comprehensive client overview dashboards
- AI-driven risk assessment and early warning systems
- Automated documentation and note generation
- Secure communication and collaboration tools

#### Clinical Support
- Evidence-based intervention recommendations
- Progress tracking and outcome analysis
- Resource library with therapeutic materials
- Group therapy facilitation tools

#### Professional Tools
- Workload optimization and scheduling assistance
- Training and development resources
- Performance analytics and quality assurance
- Secure professional networking

## 🔧 Installation

### Prerequisites
- PHP 8.x or higher
- MySQL/MariaDB database
- Web server (Apache/Nginx)
- Composer for dependency management

### Development Setup
```bash
# Clone the repository
git clone https://github.com/acesonder/NODNOL.git
cd NODNOL

# Install dependencies
composer install

# Set up database
# Configure phpMyAdmin with root/blank credentials
# Import database schema (when available)

# Configure environment
cp .env.example .env
# Edit .env with your database and security settings

# Start development server
php -S localhost:8000
```

## 🤝 Contributing

We welcome contributions from developers, healthcare professionals, and individuals with lived experience. Please read our [Contributing Guidelines](CONTRIBUTING.md) before submitting pull requests.

### Development Principles
- **Privacy First**: All features must prioritize user privacy and security
- **Evidence-Based**: Features should be grounded in addiction recovery research
- **Accessibility**: Design for users with varying cognitive and physical abilities
- **Trauma-Informed**: Consider the impact of trauma in all design decisions

## 🔐 Security

Security is paramount in this project. We implement:
- End-to-end encryption for sensitive data
- HIPAA-compliant data handling
- Regular security audits and penetration testing
- Privacy-by-design principles

If you discover a security vulnerability, please report it responsibly by emailing security@nodnol.org.
